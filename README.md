# 🚀 Bulk Student Notification System

A Laravel-based Admin Panel to manage students and send bulk notifications via Email and SMS.

---

## ✨ Features

- 📥 CSV Upload (Bulk Student Import)
- 📧 Bulk Email Sending
- 📱 Bulk SMS Sending
- 📊 Admin Dashboard
- 📈 Reports & Analytics
- 🔁 Response Tracking
- 🛠 Full CRUD Operations

---

## 🧰 Tech Stack

- Laravel (PHP)
- MySQL
- Bootstrap 5
- JavaScript
